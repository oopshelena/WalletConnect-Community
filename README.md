## 🔗 WCT Dune Dashboard
**by [gieskuy5](https://github.com/gieskuy5)**

Comprehensive on-chain analytics dashboard for the WalletConnect (WCT) ecosystem on Optimism.

**What it does:**
- Tracks staking TVL (stWCT contract) — 101.4M+ stWCT supply
- Monitors governance participation via Snapshot (4 proposals, 20K+ votes)
- Analyzes holder distribution by balance tier
- Measures daily transfer volume for stWCT & WCT tokens

**Tech stack:**
- Dune Analytics SQL queries (6 queries)
- Python data fetcher (Optimism RPC + Snapshot GraphQL)
- Chain: Optimism
